# Feature Scaling
## Normalization
$$X'=\frac{X-X_{min}}{X_{max}-X_{min}}$$
- The value all between $[0;1]$
## Standardization
$$X'=\frac{X-\upmu}{\sigma}$$
- $\upmu$ is average
- $\sigma$ is standard deviation (표준편차)
- Almost the values will be between $[-3;3]$
